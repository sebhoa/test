Les diverses techniques de positionnement en CSS (part II)

La propriété display

Cette propriété possède diverses valeurs (voir w3schools). Mais 4 sont particulièrement utilisées :

  • inline-block
  • none et block
  • flex

#01 display: inline-block

S'utilise sur un élément de type inline pour hériter des propriétés réservées aux éléments de type block notamment tout ce qui touche au dimensionnement.

#02 display: none et display: block

La valeur none permet de ne tout simplement pas afficher l'élément du tout. Alors que la valeur block permettra de l'afficher comme un élément de type block. On se sert de ce couple pour faire des menus déroulants. Les menus sont toujours organisés dans des listes (numérotées ou non cela n'a pas d'importance) elles mêmes encapsulées dans un élément nav.
